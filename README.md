# ROAD-VISION-CAR
Uma implementação de robótica móvel que substitui os tradicionais sensores infravermelhos por processamento de imagem em tempo real. Utilizando uma câmera embarcada e C++, o sistema captura a geometria da pista, calcula o centróide da trajetória e aplica algoritmos de controle em malha fechada (PID) para correção dinâmica de rota.
